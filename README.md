# Web Dev Project 2 Frontend

This frontend connects to backend auth APIs.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

Default frontend URL: `http://localhost:5173`

Frontend env values:

- `VITE_API_BASE_URL` (default: `http://localhost:5000`)

Backend folder:

- `/Users/apple/Web_Project_2/Web_Dev_Project_2_Backend`
